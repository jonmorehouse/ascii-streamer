# ASCII Streamer
Stream VOD and Live video in a terminal!

## Supported Formats

Ascii Streamer supports:

- MPEG-Dash
- Microsoft Smooth
- Apple HLS

## Usage

ascii-streamer -captions=yes -audio=no -audio-track

## Options

- captions (default no)
- stats (default no)
- bitrate (default 256)
